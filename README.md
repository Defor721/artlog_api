# ArtLog API Server

NestJS, Prisma, JWT 기반의 RESTful API 서버입니다.

## 기술 스택

- NestJS
- Prisma
- MongoDB 
- JWT (Passport 기반 인증)
- Swagger (API 문서 자동화)

## Project setup (Development)

- git clone https://github.com/Defor721/artlog_api.git
- cd artlog_api
- npm install

## Swagger

 - http://localhost:3000/api-docs
